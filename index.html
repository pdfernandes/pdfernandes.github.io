Projects

tikr

tikr provides users with real-time stock information and allows users to make trades, maintain a portfolio and manage a watchlist. tikr puts the power investment in the hands of the individual.

tikr utilizes Rails 5 for the back-end with JavaScript in the front-end. tikr leverages React and Redux to dynamically render components and manage local state.

Universtaty

Universtaty is a data visualization project written using pure JavaScript and D3. Universtaty aims to provide students with quick at-a-glance university statistics.

Universtaty uses JavaScript's asynchronous functions and DOM manipulation to retrieve university metrics from external APIs. Data is then formatted using D3 to create a variety of different charts.
